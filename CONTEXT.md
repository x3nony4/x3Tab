# x3Tab

浏览器新标签页插件。极简主题，内核为时钟、搜索、快捷启动三个模块。垂直居中布局：时钟偏上 → 搜索框中 → Docker 栏底部固定。

## Language

### 核心概念

**新标签页 (New Tab)**:
浏览器打开新标签时展示的页面，替代浏览器默认新标签页。
_Avoid_: 首页, 主页

**Docker 栏 (Dock)**:
页面底部固定展示的快捷启动栏，最多容纳 15 个快捷方式 + 1 个固定添加按钮。添加按钮始终显示。
_Avoid_: 快捷栏, 底部栏, 任务栏

**快捷方式 (Shortcut)**:
Docker 栏中的一个启动项，包含名称、URL、图标。点击在当前标签页打开对应网站（后续可配置为新标签页打开）。与浏览器自带书签无关联。
_Avoid_: 书签, 站点, 链接, 磁贴

**搜索引擎 (Search Engine)**:
搜索框当前选中的搜索目标。包含名称、URL 模板（`%s` 占位符）、颜色。以颜色圆形 + 首字母作为视觉标识。系统必须至少保留一个引擎。切换不清空已输入内容。
_Avoid_: 搜索提供方, 搜索源

**默认引擎 (Default Engine)**:
插件内置的 4 个搜索引擎：Baidu、Google、Bing、DuckDuckGo。可删除，但至少保留一个引擎（默认或自定义均可）。
_Avoid_: 内置引擎, 预置引擎

**自定义引擎 (Custom Engine)**:
用户手动添加的搜索引擎。可删除。

### 时钟

**时钟 (Clock)**:
数字时钟，显示时:分:秒。纯展示，不可点击交互。
_Avoid_: 时间

**日期 (Date)**:
时钟下方的日期显示，格式 `YYYY年mm月dd日 星期X`。
_Avoid_: 日历

### 搜索

**搜索框 (Search Bar)**:
页面中部的输入区域。左侧显示当前引擎图标，图标右侧为下拉箭头（打开引擎管理面板），右侧为输入区域。回车在当前标签页跳转搜索结果（后续可配置）。
_Avoid_: 搜索栏, 搜索输入

**引擎管理面板 (Engine Panel)**:
搜索框下方展开的面板，宽度与搜索框一致。水平排列所有引擎（图标为主、名称在图标下方为辅），末尾有 `+` 按钮添加自定义引擎。从上到下展开，带过渡动画。
_Avoid_: 搜索设置, 引擎设置, 搜索面板

**引擎切换 (Engine Switch)**:
通过 Tab 键在引擎之间循环切换。切换不清空已输入内容。
_Avoid_: 切换搜索

### 图标

**在线图标 (Online Favicon)**:
通过 MV3 `_favicon/` API 或 Google Favicon / DuckDuckGo 等第三方 API 自动获取的网站图标。
_Avoid_: 远程图标, 自动获取图标

**纯色图标 (Solid Color Icon)**:
以纯色背景 + 站点首字母（大写）生成的图标。用户可自定义颜色。
_Avoid_: 文字图标, 字母图标

**本地上传图标 (Uploaded Icon)**:
用户从本地文件系统选择的图片作为图标，存储于 IndexedDB。
_Avoid_: 自定义图片, 本地图片

### 主题

**深色主题 (Dark Theme)**:
页面背景深色、文字浅色的配色方案。系统主题不可获取时默认为此。

**浅色主题 (Light Theme)**:
页面背景浅色、文字深色的配色方案。

**主题切换 (Theme Toggle)**:
位于页面右上角的图标按钮，点击在深色 → 浅色 → 深色之间循环切换。系统主题仅用于初始化，不参与切换循环。后续设置入口也放在此位置。
_Avoid_: 主题按钮

### 交互

**编辑模式 (Edit Mode)**:
Docker 栏的编辑状态。通过右键 Docker 栏选择"编辑 Docker 栏"进入。快捷方式抖动（类似 iOS 编辑主页），右上角出现删除按钮。可长按拖拽排序。
_Avoid_: 编辑状态, 排列模式

**编辑卡片 (Edit Card)**:
快捷方式的编辑/添加弹窗。编辑模式：hover 快捷方式时显示遮罩层，包含编辑图标，点击弹出弹窗，预填现有数据。添加模式：点击 `+` 按钮弹出，表单为空。两种模式均可编辑名称（必填）、URL（必填）、图标类型（在线 / 纯色 / 上传）。
_Avoid_: 编辑弹窗, 编辑面板, 添加弹窗

**右键菜单 (Context Menu)**:
Docker 栏的右键弹出菜单。包含"编辑 Docker 栏" / "退出编辑"切换选项。进入编辑模式后才能拖拽排序和删除。
_Avoid_: 上下文菜单

**添加快捷方式 (Add Shortcut)**:
点击 Docker 栏末尾的 `+` 按钮弹出编辑卡片，填写后新增快捷方式。

### 存储

**本地存储 (Local Storage)**:
通过 `chrome.storage.local` 存储轻量设置：主题偏好、快捷方式元数据、搜索引擎配置。

**图标存储 (Icon Storage)**:
通过 IndexedDB 存储大体积数据：快捷方式图标（在线获取缓存 / 上传图片）、后续扩展的壁纸图片等。
